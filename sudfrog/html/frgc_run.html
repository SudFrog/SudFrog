frgc_run

How I'am??

I'am the module that can run the FRGC-v2 experiments (Never ask the database from my author just go to http://www.frvt.org/FRGC/ and ask).

The author could give you only the xml metadata and the masks.

I use the multi-threading to accelerate your work!

Usage:

frgc_run param queryxml targetxml inputdir maskfile Similaritymatrix

Parameters:
param Parameters xml file (see paramfile.xml )
queryxml Xml file containing the list of Query files (Test images) (see Metadata.xml example of the xml format)
targetxml Xml file containing the list of Target files (Reference images) (ame xml format as queryxml)
inputdir Path to the input geometrically normalized images (use NormFaces for that)
maskfile Mask file containing the intra/interclass tests (see MBGC mask format)
Similaritymatrix The output file containing the Similarity matrix (see MBGC similarity format)
Returns:
0 when succeed
