Detailed Description

Structure that stores the gabor filters parameters and the returned complexe part (real, imaginary, magnitude, phase).

Field Documentation

float GaborParams::dThetaOnSigma

Ratio of angular interval between filter orientations and the standard deviation of the angular Gaussian function used to construct filters in the freq. plane

int GaborParams::height

The height of the gabor filters

string GaborParams::methodname

Complexe part to be computed:

  • "REAL": real part of complex features
  • "IMAG": imaginary part
  • "MAGN": magnitude part
  • "PHASE": angle part
  • a combination example methodname="REAL, IMAG, MAGN"
