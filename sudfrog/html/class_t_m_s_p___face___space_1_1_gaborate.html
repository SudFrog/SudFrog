Detailed Description

Class that computes the gabor filtering of an input image.

It creates the log-gabor filters in frequency space based on the work of Peter Kovesi http://www.csse.uwa.edu.au/~pk/research/matlabfns/PhaseCongruency/Docs/convexpl.html
